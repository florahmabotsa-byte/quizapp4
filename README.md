 Life Hack or Urban Myth App

Life Hack or Urban Myth is a native Android app built with Kotlin in Android Studio. It helps users distinguish between real productivity tips and misleading internet myths through an interactive flashcard-style quiz.

Users are presented with statements and must decide whether each is a real life hack or an urban myth, receiving instant feedback and a final score.# quizapp4

Features

Welcome Screen
App introduction
Start button to begin quiz Flashcard Quiz
Displays a statement (life hack or myth)
Two answer buttons: Hack or Myth 
Instant feedback after each answer
“Next” button to continue
Score Screen
Displays total correct answers
Personalized feedback
Option to review answers
Review Mode
Shows all questions
Displays correct answers and explanations

Tech Stack

UI: XML Layouts
